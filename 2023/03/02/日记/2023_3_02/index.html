刘丰｜高纬智慧？也许是骗钱

发表于2023-03-02|更新于2023-04-17|感悟
|字数总计:1.3k|阅读时长:3分钟|阅读量:

摘要

最近经师姐推荐刘丰的课程，一开始听着我大吃一惊，似乎有点道理。但是随后我发现他提出的论据都是别人说、甚至很多没办法确定真伪的事情（比如习和特朗普对话什么的），这种论据我觉得根本不能支持他的论点。此外，他提到的四维默认另一个是时间，但是n维的概念应当值得是同量纲的事情，时间和空间的单位是不一样的，我总觉得他的理论是不能说服我的。如果相信他的理论，那么可能会陷入虚无主义，即找不到事物存在的意义。虽然我承认追逐外物会让自己陷入焦虑、困惑以及痛苦，但是我坚信有些东西是值得追寻的，毕竟人本质上看还是社会的一部分。我觉得这个高纬思想可能是指群体，就像蚂蚁系统组成的群体产生了部分的高纬智慧，那么人群的组成所取得的成就也必然会涉及超越单个人本身所能做到的事情。

被调情况

他是北大民营经济研究院下设社会责任与中华文化研究所的执行所长，2015年被北大撤销，随后北大宣布：**任何以“北京大学民营经济研究院”名义开展的募捐、培训、合作、讲学等活动，均属违规。**若发现，请及时联系北大。如涉及违法，请及时向当地警方报警。

民营经济研究院下设一系列研究所的人员，是可以跟民间合作的。所里人员可以不是学校编制的教授和研究员，也可以是向社会招募的办事执行人员。当然，不否认社会中有高人存在，但是听完此公言论，也能拿捏出来不是个科研人，吃开口饭还是可以。

他的一些观点

  • 看待世界的维度不同，你的认知度和自由度就不同，也就是说，生命的自由来自于意识维度的提升。

  • 爱只有在付出时才能感受到美好。当你爱一个人的时候，你专注爱到这个人的时候，你看这个世界突然变得非常的美妙，而当你开始想，他会不会同样爱我的时候，马上这个好感觉就没了。爱的根本属性是付出，就像太阳一样，他只是付出他不求回报，而且无分别。如果拿一个放大镜，把太阳光聚焦到一点，那么这一点就会烧焦，这给了我们一个非常重要的启示：一切狭隘的爱对人都是伤害。

  • 什么叫永生？到第四维没有开始没有结束，就没有生没有死了，这就是永生。所以纵向提升维度是生命的唯一方向。

  • 我问过很多成功的人，“你人生中最重要的那几个决策的依据是什么？”
    我得到的回答出奇地一致：直觉。
    直觉是什么呢？直觉是来自于高维的信息，我们人类所有的发明来自于灵感，灵感是来自高维的信息。

  • 我每个人的意识里充满了三维的像，当你脑子里把三维里最后一个像拿开的时候，你得四维智慧。

东方智慧从最高的 N 维智慧，N 趋于无穷大来看宇宙，它讲天人合一，它讲无上正等正觉，它讲唯一的神（明）。只有 N 维，N 趋于无穷大，才符合无上，才符合无极，才符合无一。

  • 我们突破三维认知只有一个功能，告诉我们这个宇宙的无限性。所以当你突破三维到达四维之后，之后的中间的维度其实全都是没有意义的，当我们执著在任何中间维度的时候，就障碍了你进入更高的维度。
  • 当你专注于一点的时候，你也可以进入这种无限的空间状态，所以不需执着在中间任何一个维度上。
  • 没有一个生命是平凡的，每一个生命内在本自聚足，聚足一切智慧。
  • 如果一个人把生命中所有的当下，都用来转念，提升，那他生活在一种全然的喜悦生命状态。所以佛教中的修行，也不是修未来，修前世，修的全是当下。
  • 所有的理论，都是让我们当下获得喜悦，当下连通高维智慧。只有当下是连接高维的。而连接高维的一瞬间产生的喜悦，是没有任何一个物质世界的喜悦能够替代的。

我的感受

  • 有些事情可能大家都知道，这个道理比如“事物的底层逻辑比事物更核心”，都是他只提到观点却不给出具体的方法论
  • 提到构成世界的根本是“波动”，但是没有诠释生命的意义。如果真的都是那么简单，我们存在的意义是什么？
